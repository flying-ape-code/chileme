## 🐛 Bug 管理

### Bug 分级
- **P0:** 阻塞发布 (立即修复)
- **P1:** 严重问题 (24 小时内)
- **P2:** 一般问题 (本周内)
- **P3:** 优化建议 (后续迭代)

### Bug 提交流程
1. 发现 Bug → 截图/录屏
2. 创建 GitHub Issue
3. 标记 `type:bug` + 优先级
4. 分配给 dev agent
5. 验证修复 → 关闭 Issue
