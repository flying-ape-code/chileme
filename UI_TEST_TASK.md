## 🐛 Bug 提交流程

### 发现 Bug 后

1. **记录 Bug 信息**
   - Bug 描述
   - 复现步骤
   - 预期行为 vs 实际行为
   - 截图（如果可能）
   - 浏览器控制台错误
   - 测试环境信息

2. **创建 Issue 文件**
   - 路径: `~/chileme/issues/`
   - 文件名格式: `issue-[ID]-[SHORT_TITLE].md`
   - ID 格式: `001`, `002`, `003`（递增）
   - 使用 issues/.template.md 作为模板

3. **Issue 示例**

文件名: `~/chileme/issues/issue-001-register-fails.md`

```markdown
# Issue: 注册失败 - 无效邮箱格式未显示错误

**Issue ID:** 001
**Created:** 2026-03-11 23:50
**Status:** Open

---

## 🔍 Bug Description

在注册页面输入无效的邮箱格式（如 "invalid-email"），点击注册按钮后，没有显示错误提示，页面无响应。

---

## 📋 Steps to Reproduce

1. 访问 http://localhost:5173/register
2. 用户名输入: `testuser`
3. 邮箱输入: `invalid-email`
4. 密码输入: `test123`
5. 确认密码输入: `test123`
6. 点击"注册"按钮

---

## 🎯 Expected Behavior

应该显示错误提示："请输入有效的邮箱地址"，并阻止注册提交。

---

## ❌ Actual Behavior

没有显示错误提示，页面无任何响应，用户不知道发生了什么。
```
