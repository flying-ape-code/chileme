## 🗄️ 数据库表结构

### meals 表

| 字段 | 类型 | 说明 |
|------|------|------|
| id | UUID | 主键 |
| name | VARCHAR(255) | 商品名称 |
| category | VARCHAR(50) | 分类 |
| image_url | TEXT | 图片 URL |
| cps_link | TEXT | CPS 推广链接 |
| price_min | DECIMAL(10,2) | 最低价格 |
| price_max | DECIMAL(10,2) | 最高价格 |
| rating | DECIMAL(3,2) | 评分（0-5） |
| distance | VARCHAR(50) | 距离 |
| delivery_time | VARCHAR(50) | 配送时间 |
| is_active | BOOLEAN | 是否上架 |
| sort_order | INTEGER | 排序权重 |
| click_count | INTEGER | 点击次数 |
| conversion_count | INTEGER | 转化次数 |
| created_at | TIMESTAMP | 创建时间 |
| updated_at | TIMESTAMP | 更新时间 |

### 索引
- `idx_meals_category` - 分类查询优化
- `idx_meals_active` - 活跃商品过滤优化
- `idx_meals_sort` - 排序优化

### RLS 策略
- **public_view_active:** 所有人可查看活跃商品
- **admin_all:** 管理员可管理所有商品
