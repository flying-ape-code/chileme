## 实施方案

### 方案 1: App.tsx 导航栏优化

**问题:** 当前导航栏在小屏幕上过于拥挤（8+ 个按钮）

**解决方案:**
1. 移动端使用汉堡菜单 + 底部 TabBar
2. 桌面端保持现有导航
3. 添加当前页面高亮

```tsx
// 移动端导航结构
<header>
  ☰ 吃了么                    👤
</header>

<bottom-tab-bar>
  [首页] [历史] [💬] [我的] [更多]
</bottom-tab-bar>
```

### 方案 2: 对话框优化

**问题:** 
- ShareModal 没有关闭按钮
- 点击外部无法关闭
- 没有操作反馈

**解决方案:**
1. 所有模态框添加统一的关闭按钮
2. 点击 backdrop 关闭
3. 添加 Toast 提示系统

### 方案 3: 响应式布局增强

**断点:**
- 超小屏：320px - 375px
- 小屏：375px - 640px
- 中屏：640px - 1024px
- 大屏：1024px+

**优化点:**
1. 字体大小自适应（使用 clamp）
2. 按钮最小尺寸 44px
3. 对话框最大高度 80vh，内部可滚动
4. 安全区域适配（iPhone X+）
