# 用餐喜好设置使用指南

**Issue:** #48  
**版本:** 1.0  
**上线日期:** 2026-04-05

---

## 📱 功能介绍

用户可以设置用餐喜好，转盘会根据喜好进行个性化推荐。

---

## 🎯 喜好类型

### 1. 饮食类型（单选）
- 🍽️ 杂食 - 什么都吃
- 🥩 肉食 - 偏好肉类
- 🥬 素食 - 只吃素食

### 2. 口味偏好（多选）
- 🌶️❌ 不辣
- 🌶️ 微辣
- 🌶️🌶️ 辣
- 🌶️🌶️🌶️ 重辣

### 3. 喜欢的食材（多选）
- 🐄 牛肉
- 🐷 猪肉
- 🐔 鸡肉
- 🐑 羊肉
- 🦐 海鲜
- 🥬 蔬菜
- 🥚 蛋类
- 🧀 奶制品

### 4. 忌口设置（多选）
- 🌿 香菜
- 🧅 葱
- 🧄 蒜
- 姜
- 🌶️ 辣椒
- 🫘 芝麻
- 🥜 花生
- 🦪 贝类

---

## 🔧 权重算法

| 类型 | 权重 | 说明 |
|------|------|------|
| 忌口 | 0 | 完全排除 |
| 不喜欢 | 0.3 | 降低概率 |
| 一般 | 1.0 | 正常概率 |
| 喜欢 | 2.0 | 增加概率 |

---

## 📍 入口位置

**个人中心 → 用餐喜好**

---

## ⚠️ 边界情况

### 1. 新用户无喜好设置
- 行为：使用全部商品
- 提示：无

### 2. 筛选后商品<3 个
- 行为：显示警告
- 提示："筛选后商品较少，建议放宽喜好设置"

### 3. 所有商品都被忌口
- 行为：显示错误
- 提示："所有商品都被忌口，请清空部分忌口设置"
